Basic Multi-Objective Problem:

f₁(x) = x[0]**2 + x[1]**2

f₂(x) = (x[0]-1)**2 + x[1]**2

Simple convex problem with competing objectives
Engineering Design Problem:

f₁(x) = 2 + (x[0] - 2)**2 + (x[1] - 1)**2

f₂(x) = 9*x[0] - (x[1] - 1)**2

Represents cost and performance trade-off
Portfolio Optimization:

f₁(x) = -(0.6*x[0] + 0.4*x[1])

f₂(x) = x[0]**2/6 + x[1]**2/2

Maximize return (-f₁) while minimizing risk (f₂)

Performance Metrics

Metrics Explanation:
  • GD (Generational Distance): Measures how close the obtained solutions are to the true Pareto front. Lower values indicate better convergence.
  • HV (Hypervolume): Measures both convergence and diversity of solutions. Higher values indicate better overall performance.
